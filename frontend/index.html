How it works:

  1. Select a file from your computer
  2. The file is encrypted in your browser using AES-256-GCM
  3. Only the encrypted data is uploaded to the server
  4. You receive a shareable link with the decryption key
  5. Anyone with the link can download and decrypt the file

Note: Files expire after 24 hours by default. The server NEVER sees your encryption key (zero-knowledge).
